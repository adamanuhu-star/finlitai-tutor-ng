# 💰 FinLitAI Tutor NG 🇳🇬

**Smart Money Assistant for Nigerians**

FinLitAI Tutor NG is an AI-powered financial literacy assistant designed to help Nigerians make smarter financial decisions, avoid scams, and build better money habits.

---

## 🚀 Problem

In Nigeria, many young people lack basic financial knowledge.  
This leads to:
- Poor saving habits  
- Bad financial decisions  
- Falling victim to scams and Ponzi schemes  

---

## 💡 Solution

FinLitAI Tutor NG provides simple, accessible financial education using:
- **English**
- **Nigerian Pidgin**

Making it easy for everyone to understand and apply in real life.

---

## ✨ Features

- 💬 AI-powered chat assistant  
- 🚨 Scam detection and alerts  
- 💡 Daily financial tips  
- 💰 Savings calculator  
- 📚 Financial lessons (Savings, Budgeting, Banking)  
- ⚠️ Offline backup mode (works without internet)  

---

## 🧠 How It Works

- Built with **Streamlit** for the interface  
- Uses **AI API** for intelligent responses  
- Includes **fallback system** for offline usage  

---

## 🇳🇬 Nigerian Context

FinLitAI is tailored for real-life Nigerian scenarios:
- Ajo (local savings system)  
- POS transactions  
- Betting risks  
- Ponzi schemes  
- Fake crypto investments  

---

## 🎯 Impact Goal

To empower **1 million+ Nigerian youths** with financial knowledge and help them:
- Save smarter  
- Avoid scams  
- Build financial independence
